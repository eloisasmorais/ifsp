Algumas Tecnologias Associadas ao Desenvolvimento Web
HTML HTML(abreviação para expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web.
CSS Cascading Style Sheets(CSS) é um mecanismo para adicionar estilos (cores, fontes, espaçamento etc) a um documento, o CSS cria um link (ligação) para uma página que contém estilos.
JavaScript Internet Explorer é uma série de navegadores web gráficos desenvolvidos pela Microsoft e inclusos como parte da linha de sistemas operacionais Microsoft Windows, iniciado em 1995.
PHP PHP (um acrônimo recursivo para "PHP: Hypertext Preprocessor") é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.
