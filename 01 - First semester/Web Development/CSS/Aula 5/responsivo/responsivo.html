Media Query

Cada página deve ser criada com um layout específico, para que o navegador obedeça as regras e configure de acordo com a necessidade

Media Query é uma técnica introduzida no CSS3. Ela usa a regra @media para incluir um bloco de propriedades CSS somente se uma determinada condição for verdadeira.
